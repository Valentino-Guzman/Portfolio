Sistema de Gestión de Mantenimiento

Angular Express.js

Plataforma web desarrollada para la Universidad Tecnológica Nacional que permite gestionar órdenes de mantenimiento en tiempo real. Los usuarios pueden crear, editar y seguir el estado de cada orden, asignar responsables y consultar el historial de intervenciones. El sistema está construido con Angular en el frontend, Express.js en el backend y utiliza SCSS para una interfaz moderna y adaptable.

Prolytics – Plataforma de Análisis de Rendimiento para League of Legends

Angular Express.js

Prolytics es una plataforma que permite analizar el rendimiento de jugadores en League of Legends. La aplicación presenta estadísticas detalladas, gráficos visuales y rankings, optimizando la experiencia de usuario con un diseño moderno y responsivo. Este proyecto fullstack combina llamadas a la API de Riot Games, procesamiento de datos y visualización atractiva, enfocándose en brindar insights claros para jugadores competitivos.
